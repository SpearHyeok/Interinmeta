# 결제 모듈 프로젝트

## 프로젝트 개요

- 슈킹의 결제 모듈에 관한 컴포넌트입니다.
- 카드 등록, 결제 기능을 지공합니다.
- 만만두, hee, rlaehgus

## 개발 환경

 react: 18.3.1
 node: 20.15.0
 JavaScript ES6

## 설치 및 실행 방법
```
npm install
npm start
```

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할

- `CardForm`: 신용카드 정보를 담을 것
- `CardInput` : 신용카드 정보를 입력할 페이지
